# Crawl
 Crawl data contest

## 1. Crawl data from website
- Crawl data contest from website: https://www.coder.husc.edu.vn/

## 2. Using
- Login and get cookies in website: https://www.coder.husc.edu.vn/
- Edit file key.json
```json
{
   "csfrtoken": "your_csfrtoken",
   "sessionid": "your_sessionid"
}
```

## 3. Run
- Run file main.py
